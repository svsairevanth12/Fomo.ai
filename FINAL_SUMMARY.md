## ⚠️ Important: Before Running

### Fix Node.js PATH (Required on Windows)

**Quick Steps:**
1. Press `Windows + R`
2. Type `sysdm.cpl` → Enter
3. Advanced → Environment Variables
4. Edit "Path" → Add: `C:\Program Files\nodejs\`
5. OK → Restart terminal
6. Test: `node --version`

**Then:**
```bash
npm install
npm run electron:dev
```
